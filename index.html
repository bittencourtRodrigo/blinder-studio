Blinder Studio

A personal command-line tool that schedules YouTube Shorts uploads from the developer's own laptop. Single user, single machine, no hosted service.

What it does

Blinder Studio runs once a night on the developer's laptop. For each YouTube brand channel the developer owns, it produces a short-form video locally (script generation, voice synthesis, stock footage assembly via FFmpeg) and uploads it to YouTube with a future publishAt time so the video appears on the channel the next day.

Who can use it

Only the developer who runs the source code locally. There is no signup, no public deployment, no shared instance. The OAuth client is configured for the developer's own Google account, which owns the YouTube brand channels the tool uploads to.

What data it touches

  • OAuth refresh tokens for the YouTube Data API youtube.upload scope. Stored locally on the developer's laptop in plain JSON files inside a credentials/ directory that never leaves the machine.
  • Video metadata (title, description, tags, publish time) generated on the laptop and sent only to YouTube as part of each upload.

See the Privacy Policy for the full description, including how data is used, retained, and how to revoke access.

Distribution

Blinder Studio is a personal automation maintained by its developer for their own YouTube channels. It is not offered as a hosted service, a downloadable product, or a public installation.
